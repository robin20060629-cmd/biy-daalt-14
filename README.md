# Бие даалтын тайлан (Reflection)

## 1. Сонгосон сэдэв
**Сэдэв:** DummyJSON API-д суурилсан автоматжуулсан тест болон CI/CD интеграци.
**Зорилго:** Цахим худалдааны (E-commerce) системийн Backend-ийн үндсэн функцууд болох хэрэглэгчийн нэвтрэлт (Authentication), бараа бүтээгдэхүүний удирдлага (Products) болон алдааны тохиолдлуудыг (Error handling) Postman болон Newman ашиглан шалгах.

## 2. Гүйцэтгэсэн ажил
* **API Endpoints:** Дараах чиглэлээр 10 гаруй request бэлдсэн:
    * **Auth:** Нэвтрэх болон Token авах үйлцэл.
    * **Products:** Бараа хайх, шинээр нэмэх, засах, устгах (CRUD).
    * **Error Cases:** Буруу ID-гаар хайх, эрхгүй үед хандах гэх мэт.
* **Automation:** Newman ашиглан терминалаас тест ажиллуулж, HTML тайлан үүсгэсэн.
* **CI/CD:** GitHub Actions ашиглан `.github/workflows/api-tests.yml` тохиргоог хийж, автоматжуулалтыг дуусгасан.

## 3. Дүгнэлт
Энэхүү бие даалтаар орчин үеийн програм хангамжийн тестийн чухал хэсэг болох API тестийг гар аргаар болон автоматаар хэрхэн гүйцэтгэхийг практик дээр сурлаа. Мөн GitHub Actions ашиглан кодын чанарыг тогтмол хянах (Continuous Integration) туршлага хуримтлуулсан.
